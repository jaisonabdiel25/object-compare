# Object Compare (JSON Sorter)

Script en **Python** que ordena objetos JSON por el **nombre de sus propiedades**, de forma **recursiva**, incluyendo:
- objetos anidados
- objetos dentro de listas

Ideal para normalizar JSON, comparar estructuras o preparar datos antes de enviarlos o guardarlos.
